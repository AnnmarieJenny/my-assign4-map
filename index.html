Publicly-Owned Vacant Lots in
New York City

This map seeks to identify publicly-owned vacant lots in NYC. It currently uses the NYC PLUTO Dataset. One might note the odd prevalence of vacant lots displayed on the map. To remedy this, this map will soon incorporate data from NYC OpenData. This will provide the map with more avenues to refine the filter so the map only shows currently vacant, publicly-owned lots.

*Please note this map displays only a rough estimation of vacant lots and requires further cleaning to ensure accuracy.

For more information on this dataset, visit NYC PLUTO
